# دليل المحاضرة 7

تغطي هذه المحاضرة موضوعات متقدمة في **الحلقات التكرارية (Loops) والقوائم المتداخلة** بالإضافة إلى **استيراد المكتبات**، وتشمل:
1. **القوائم المتداخلة**: تخزين قوائم داخل قائمة رئيسية لتنظيم البيانات المعقدة (مثل تخزين اسم الطالب ودرجته معا).
2. **التحكم بالحلقات (`break` و `continue`)**: استخدام أمر `break` لكسر الحلقة فوراً عند إيجاد الهدف، واستخدام أمر `continue` لتجاهل الكود في الدورة الحالية والانتقال الفوري للدورة التي تليها (مثل تجاوز الأرقام الفردية).
3. **الجملة الشرطية التابعة للحلقات (`for...else`)**: تنفيذ كود معين (مثل رسالة "لم يتم العثور") فقط إذا اكتملت الحلقة دون أن يتم كسرها بأمر `break`.
4. **حلقة `while` والتكرار اللانهائي**: إضافة عدادات لحلقة `while` لضمان عدم حدوث تكرار لانهائي، وكذلك كيفية استخدام الحلقات اللانهائية المتعمدة `while True` مع أمر الكسر بناء على شرط إنهاء من المستخدم (مثل إدخال حرف معين).
5. **استيراد المكتبات (`import`)**: كيفية استخدام الأكواد والمكتبات الجاهزة عبر أمر `import`، مثل مكتبة `webbrowser` لفتح المواقع الإلكترونية.
