# دليل المحاضرة 6

تغطي هذه المحاضرة موضوع **القوائم (Lists) والحلقات التكرارية (For Loops)** في بايثون، وتشمل:
1. **إنشاء القوائم والوصول لعناصرها**: إنشاء قائمة باستخدام الأقواس المعقوفة `[]`، ومعرفة طولها بدالة `len()`، والوصول للعناصر عبر الفهارس (Indexing) والتقطيع (Slicing).
2. **إضافة وحذف العناصر**: استخدام دوال مثل `append()` و `insert()` و `extend()` للإضافة، ودوال مثل `remove()` و `pop()` للحذف.
3. **التعديل والترتيب والبحث**: تعديل العناصر عبر الفهرس مباشرة، وترتيب القائمة باستخدام `sort()`، ومعرفة عدد تكرار عنصر بدالة `count()`.
4. **الدوال الحسابية والاستنساخ**: حساب المجموع `sum()`، أقل قيمة `min()`، وأكبر قيمة `max()`. إضافة لفهم الفرق بين مساواة القوائم (مرجع لنفس الذاكرة) ونسخها للاستقلالية باستخدام `copy()`.
5. **الحلقات التكرارية (`for`)**: المرور على عناصر القائمة لاستخراج قيمها تباعا، أو استخدام دالة `range()` للتكرار بعدد محدد، مع دمج الشروط `if` بداخلها كطريقة للتحقق من الأعداد الفردية والزوجية.
