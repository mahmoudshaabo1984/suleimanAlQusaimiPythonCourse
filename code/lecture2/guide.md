# دليل المحاضرة 2

تغطي هذه المحاضرة الأساسيات في لغة بايثون، وتشمل:
1. **دالة الطباعة (`print`)**: كيفية طباعة النصوص والأرقام على الشاشة.
2. **المتغيرات (Variables)**: كيفية تخزين البيانات مثل النصوص (`strings`) والأرقام (`integers`) داخل متغيرات واستخدامها.
3. **العمليات الحسابية الأساسية**: كيفية إجراء عمليات الجمع بين الأرقام والمتغيرات.
4. **دالة الإدخال (`input`)**: كيفية استقبال البيانات من المستخدم عبر لوحة المفاتيح وكيفية تحويل النصوص المدخلة إلى أرقام صحيحة باستخدام دالة `int()`.
