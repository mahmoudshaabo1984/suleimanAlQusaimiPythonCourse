# دليل المحاضرة 8

تغطي هذه المحاضرة المشاريع العملية والمولدات العشوائية ومقدمة في الدوال التعريفيّة:
1. **التعامل مع العشوائية (`random`)**: استيراد الدالة من مكتبة العشوائية لتوليد أرقام عشوائية مسيجة `randint`، أو لاختيار عنصر عشوائي محدد من نص أو قائمة `choice`.
2. **برنامج لعبة تخمين الأرقام**: بناء لعبة متكاملة تعتمد على العشوائية وحلقات `while` لإرشاد المستخدم ذكيا (ما إذا كان الرقم أكبر أو أصغر من الرقم السري) مع عدّاد للمحاولات المسموحة.
3. **الصوتيات في بايثون (`winsound`)**: استخدام مكتبة `winsound` المدمجة في نظام ويندوز لإضافة المؤثرات الصوتية للألعاب عند الفوز أو الخسارة.
4. **برنامج توليد كلمات المرور**: استخدام دالة `choice` داخل حلقة تكرارية لبناء كلمة مرور قوية عشوائية ذات طول معين يحدده المستخدم.
5. **مقدمة إلى الدوال (`def`)**: كيفية إنشاء دالة مخصصة (Custom Function) باستخدام الكلمة المفتاحية `def`، وكتابة كود شامل يعيد قيمة فعلية باستخدام `return` (مثل بناء دالة مخصصة لحساب الطول).
